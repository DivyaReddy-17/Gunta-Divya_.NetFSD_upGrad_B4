How to run the app:
-------------------
1. Open index.html directly in a web browser (no build step required)
2. Or use a local server: e.g. npx serve . (optional)

How to run the tests:
--------------------
1. Open a terminal in the project root
2. Run: npm install
3. Run: npm test
